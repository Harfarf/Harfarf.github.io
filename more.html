Unity C# - CompareTag

Why should we use 'CompareTag' instead of '==Tag' ?
• Performance: string comparisons are slower, less internally optimized and less precise.
• Error Handling: CompareTag has a built-in check to ensure the tag exists, and if not Unity will throw an error.

Data Protocols used in multiplayer games:

Multiplayer Protocol P2P

• There is no central server. All clients are connected to each other.
• Each client shares data directly with the others.
• Advantages: Less expensive, no central server.
• Disadvantages: Less secure and there may be synchronization issues.
• Examples: Terraria, Payday2, Dark Souls

Host dependency: If the host player leaves, the game closes.
Host lag: If the host has a bad connection, it affects all players.
Host migration: In some games (such as Call of Duty), if the host leaves, the game pauses and attempts to assign a new host.

Multiplayer Protocol Client-Server

• There is a central server that manages all the information.
• Clients (players or users) connect to the server, but not to each other.
• Advantages: More secure and easier to control.
• Disadvantages: If the server fails, everything stops working.
• Examples: Roblox, Fortnite, LoL, WoW, Overwatch

No Host dependency: Game continues even if the host disconnects.
No lag: Statistics and progress are well synchronized.
